  • 個人向けMicrosoft 365（Personal/Family）の大幅値上げ：
    • Personal：月額1,490円 → 2,130円（約43%値上げ）、年額13,410円 → 21,300円
    • Family：月額2,100円 → 2,740円（約30%値上げ）、年額21,000円 → 27,400円
    • Copilot機能が統合されたが、月60ポイントのAIクレジット制で使い放題ではない
    • Copilot不要な場合は「クラシックプラン」への変更が期間限定で可能
Microsoft Build 2025での主な発表内容：
  • Microsoft 365 Copilot Tuning：企業が自社のデータ、ワークフロー、プロセスを使用してAIモデルをファインチューニングできる新機能。法律事務所の契約書作成や企業固有の質問応答など、専門的なタスクの精度向上が可能に
  • マルチエージェント オーケストレーション：複数のAIエージェントが協調して複雑なワークフローを自動化（例：新人オンボーディングで施設・財務・法務部門のエージェントが連携）
  • GitHub Copilot統合強化：VS CodeのGitHub Copilot拡張機能のオープンソース化
  • Model Context Protocol (MCP) サポート：WindowsがMCPをネイティブサポート
  • これらの機能は2025年6月以降、段階的にリリース予定（一部は5000ライセンス以上の大企業向け）
導入検討時の注意事項：
  • Microsoft 365 Copilot Chat（無償版）は、基本的なAIチャット機能のみで、Officeアプリとの統合や組織データアクセスは不可
  • 有償版は組織の生産性向上に大きく貢献するが、月額4,722円（税抜）の追加コストが必要
  • 2025年4月から法人向けライセンスも約20%値上げされたため、総コストの検討が重要
  • Copilot Tuningなどの最新機能は大企業向けに先行提供される傾向がある
  • 個人向けプランは値上げと同時にCopilot統合されたが、ポイント制限があるため注意
